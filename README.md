 # My project

 ## Lab 3 assignment

## Project Description
This is a responsive portfolio website created using, HTML, CSS(Flexbox, Grid), and animation.

## Features
- Responsive layout
- hover effect
- animation
- CSS Grid projects section

## How to run
Open "portfolio.html" in any browser.

## how to view the website

1. Open the project folder and run
   portfolio.html in any browser.
2. Right-click → Inspect → Toggle Device Toolbar
3. Check views for:
   a. Mobile (360–480px)
   b. Tablet (768px)
   c. Desktop (1024px+)
4. Test animations & interactions:
   Navigation hover
   Button hover
   Card hover scaling
   Hero text color-changing animation

## Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media queries
